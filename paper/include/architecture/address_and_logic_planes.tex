\subsection{The Two-Tiered Universe: Address Plane and Logic Plane}
\label{sec:dual_architecture}

The architecture introduces a fundamental bifurcation between static persistent storage and volatile generative reasoning. Rather than a monolithic neural network, the system operates across two interacting structural planes, strictly separating the human-facing input/output symbols from the machine's internal reasoning substrate.

\subsubsection{The Address Plane (LSM): The Static Ground}
The Address Plane functions as the deterministic, append-only long-term memory of the system, implemented as a Morton-keyed Log-Structured Merge (LSM) tree. It eschews traditional continuous embeddings in favor of a \textbf{Thermodynamic Trie}. 

Crucially, \textbf{the (input and output) value IS the key, and the stored value is the link to the system's native value type.} Specifically, every input byte represents the X-coordinate of a Morton key, and its boundary-local depth (which resets at sequencer boundaries) forms the Y-coordinate. Because the byte value is explicitly encoded as the spatial coordinate itself, the stored 512-bit Value at that coordinate is completely free of lexical identity. Instead, the stored Value acts as a local computational operator---a native element of the $\operatorname{GF}(257)$ Fermat field that carries affine phase, routing hints, and rotational momentum. By resetting the local depth at each structural boundary, identical byte sequences geometrically collapse onto the same coordinate, making \emph{collision} synonymous with \emph{compression}.

\subsubsection{The Logic Plane: Volatile Reasoning and Ephemeral Registers}
In contrast to the static Address Plane, the Logic Plane provides a volatile, high-dimensional scratchpad explicitly designed for associative reasoning and BVP cantilever solving. When prompted, the system spins up a dynamic graph driven by a multi-headed wavefront. This wavefront carries \textbf{Ephemeral Execution Registers}---transient state arrays and affine transforms that exist only to seed backtracking and interpolation, without ever being persisted into the LSM.

Reasoning in the Logic Plane diverges significantly from auto-regressive next-token prediction, treating computation as a process of geometric resonance and affine group rotation over $\operatorname{GF}(257)$. The execution lifecycle aligns with this logic:
\begin{enumerate}[nosep]
    \item \textbf{Excite:} The prompt's input bytes are converted into initial phase seeds, injecting measurable momentum into the system.
    \item \textbf{Rotate:} The ephemeral execution registers apply non-destructive affine rotations $f(x) = (ax + b) \pmod{257}$, advancing the state vector through the field. Rotation \emph{is} execution.
    \item \textbf{Measure:} The newly computed phase is measured against stored anchors in the Address Plane via bitwise XOR. Constructive interference (phase-locking) produces a continuation, while destructive interference pruning triggers the \textbf{Frustration Engine} to pivot the orbit.
    \item \textbf{Halt \& Project:} The process repeats until reaching a fixed point or saturation horizon. The final coordinates are then un-Mortoned (reversing the coordinate back to the byte key) to translate the answer back into human-facing semantics.
\end{enumerate}
