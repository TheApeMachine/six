\subsection{Analytical Phase Model}
\label{sec:eigenmode_cooccurrence}
\label{sec:eigenmode_navigation}

Unlike spectral methods that require building co-occurrence matrices and
extracting eigenvectors, the phase model derives toroidal coordinates
\emph{analytically} from the intrinsic bit distribution of each value over
$\operatorname{GF}(8191)$.  No training data is required; every value's position
on the phase torus is determined at construction time.

\subsubsection{Phase Assignment via Prime Geometry}

Each value $\mathbf{c}$ has $k = \|\mathbf{c}\|_1$ active bits at positions
$\{p_1, \ldots, p_k\} \subset \{0, \ldots, 8190\}$.  We assign toroidal
coordinates $(\theta, \phi) \in [-\pi, \pi)^2$ directly:

\paragraph{Theta (Circular Mean of Active Positions).}
The angular identity of a value is the circular mean of its active prime
indices, uniformly distributed around $[0, 2\pi)$ over the 8191-element field:

\begin{equation}
  \theta(\mathbf{c}) = \operatorname{atan2}\!\Bigl(
    \sum_{j=1}^{k} \sin\!\bigl(\tfrac{2\pi \, p_j}{8191}\bigr),\;\;
    \sum_{j=1}^{k} \cos\!\bigl(\tfrac{2\pi \, p_j}{8191}\bigr)
  \Bigr)
  \label{eq:theta_analytical}
\end{equation}

This phase is geometrically equivalent to a $\operatorname{RotateX}$
translation on the Fermat Cube: values with similar prime activation patterns
occupy nearby $\theta$-positions on the torus.

\paragraph{Phi (Density Phase).}
The secondary phase encodes the value's population density:

\begin{equation}
  \phi(\mathbf{c}) = \frac{2\pi \cdot \|\mathbf{c}\|_1}{8191}
  \label{eq:phi_analytical}
\end{equation}

Sparse values (low popcount) sit near $\phi = 0$; dense values approach
$\phi = 2\pi$.  The pair $(\theta, \phi)$ embeds every value onto the
2-torus $\mathbb{T}^2 = S^1 \times S^1$.

\subsection{Sequential Phase Tracking and Intent Alignment}
\label{sec:toroidal_steering}

During generation, the accumulated sequential phase is tracked as a
\emph{weighted circular mean} over the emitted value sequence
$(\mathbf{c}_1, \ldots, \mathbf{c}_n)$:

\begin{equation}
  \bar{\theta}_n = \operatorname{atan2}\!\Bigl(
    \sum_{i=1}^{n} w_i \sin \theta(\mathbf{c}_i),\;\;
    \sum_{i=1}^{n} w_i \cos \theta(\mathbf{c}_i)
  \Bigr)
  \label{eq:seq_phase}
\end{equation}

where $w_i = \|\mathbf{c}_i\|_1$ weights each value by its active count
(density).  The \emph{concentration parameter} $\kappa_n$ (analogous to the
$\kappa$ parameter of the von Mises distribution) measures phase coherence:

\begin{equation}
  \kappa_n = \frac{\sqrt{S_n^2 + C_n^2}}{W_n}, \qquad
  S_n = \sum_{i=1}^{n} w_i \sin \theta_i, \quad
  C_n = \sum_{i=1}^{n} w_i \cos \theta_i, \quad
  W_n = \sum_{i=1}^{n} w_i
  \label{eq:concentration}
\end{equation}

Values of $\kappa_n$ near 1 indicate strong phase alignment (the trajectory
is coherent); values near 0 indicate phase diffusion (semantic drift).

\subsection{Geometric Closure Detection}
\label{sec:geometric_closure}

Generation terminates when the candidate value sequence's weighted circular
mean phase returns to within a configurable threshold $\delta$ of the anchor
phase $\bar{\theta}_{\text{anchor}}$ established at prompt initialization:

\begin{equation}
  \Delta\theta = \min\bigl(|\bar{\theta}_n - \bar{\theta}_{\text{anchor}}|,\;\;
                           2\pi - |\bar{\theta}_n - \bar{\theta}_{\text{anchor}}|\bigr)
  < \delta
  \label{eq:closure}
\end{equation}

where $\delta$ is derived from the architecture's Shannon capacity parameter
(\texttt{shannonCapacity}, default $0.45$, a density threshold).  This constitutes a
topological proof of structural closure: the generative trajectory has
completed a semantically coherent loop on the torus.  Because $\mathbb{T}^2$
is a compact manifold, every trajectory is pre-compact (bounded), preventing
entrapment in isolated fixed points or periodic orbits.

\subsection{Advantages of the Analytical Model}

\begin{itemize}[nosep]
  \item \textbf{Zero training cost.}  No co-occurrence matrices, no
        eigendecomposition, no corpus-dependent calibration.  The phase map
        is a pure function of the value's bit pattern.
  \item \textbf{Deterministic.}  Identical values always map to identical
        $(\theta, \phi)$ regardless of context, ensuring reproducibility.
  \item \textbf{$O(k)$ per value.}  Phase computation scales with the number
        of active bits ($k \leq 8191$), not with corpus size.
  \item \textbf{Intrinsically geometric.}  The prime-index circular mean
        directly corresponds to the value's position in the
        $\operatorname{GF}(8191)$ rotational lattice, making phase navigation
        and Galois field operations composable.
\end{itemize}
