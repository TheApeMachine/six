\subsection{Structural Layout: The Address Plane}
\label{sec:manifold_layout}
\label{sec:manifold_architecture}

The architecture utilizes a unified address plane in the form of a Morton-keyed Log-Structured Merge (LSM) index. A cell address packs the byte identity ($X$) and the boundary-local depth ($Y$) into a single 64-bit key:

\begin{equation}
  \text{CellKey} = \text{Pack}(\underbrace{v}_{\text{byte value (X)}},\; \underbrace{d}_{\text{local depth (Y)}})
  \label{eq:morton_pack}
\end{equation}

The primary layout is computationally efficient ($\texttt{symbol} \ll 32 \mid \texttt{localDepth}$), ensuring that data scan locality intrinsically follows the underlying sequential stream. Because the depth $d$ resets to 0 at each structural boundary discovered by the sequencer, distinct sequences that share identically situated local depths collapse geometrically onto the exact same cell coordinate. 

\subsection{The 512-Bit Native Monotype}
\label{sec:native_monotype}

At each defined Morton coordinate, the system instantiates its fundamental computational primitive: a 512-bit \emph{Value}. Unlike traditional symbol embeddings, the Value does not redundantly encode the byte (which is already encoded by the spatial $X$ coordinate), but rather serves as a self-contained operational state.

The 512 bits are structurally partitioned to separate the generative logic from routing metadata:
\begin{itemize}[nosep]
  \item \textbf{Bits 0--255:} The $\operatorname{GF}(257)$ Fermat core. This acts as the mathematical reasoning state, participating in core affine evolutions.
  \item \textbf{Bit 256:} The delimiter and algorithmic halt flag, terminating generative loops.
  \item \textbf{Bits 257--511:} The Operator Shell. These upper bits operate beyond the $\operatorname{GF}(257)$ plane and carry affine constants, route hints, trajectory snapshots, and mutation limits (guard radii).
\end{itemize}

This layout is strictly aligned for 512-bit hardware constraints, ensuring zero-cost metadata propagation alongside the core mathematical updates.

\subsection{Generative Coordinate Transforms}
\label{sec:affine_transform}

With the static sequence completely decoupled into the LSM coordinates, reasoning progresses strictly through generative transform logic within the Fermat core array. The core performs affine phase rotation over the cyclic $\operatorname{GF}(257)$ group:

\begin{equation}
  f(x) = (a \cdot x + b) \pmod{257}, \qquad
  a \in \{1, \ldots, 256\},\; b \in \{0, \ldots, 256\}
  \label{eq:gf_affine}
\end{equation}

By composing sequences of these rotations, the wavefront maps logical deductions strictly into deterministic spatial leaps across the Morton-keyed address plane.
